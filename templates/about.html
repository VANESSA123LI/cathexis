About Cathexis

What is Flask?

Flask is a lightweight web framework for Python. It's called a "microframework" because it doesn't require particular tools or libraries. This makes it:

  • Simple: Easy to learn and use
  • Flexible: You choose what you need
  • Powerful: Can build complex applications
